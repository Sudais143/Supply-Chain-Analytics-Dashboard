# Supply Chain Analytics Dashboard - Adom Retail Ltd

![Power BI](https://img.shields.io/badge/Power%20BI-F2C811?style=for-the-badge&logo=powerbi&logoColor=black)
![Status](https://img.shields.io/badge/Status-Complete-success?style=for-the-badge)

> **Integrated Business Intelligence Dashboard for Supply Chain Management**

A Power BI dashboard solution analyzing 13 months of operational data (March 2024 - March 2025) across inventory, sales, fulfillment, and replenishment functions for a mid-sized retail company in Ghana.

---

## 📊 Dashboards

| Dashboard | Key Metrics |
|-----------|-------------|
| **Inventory Management** | Stock levels, transaction types, warehouse distribution, inflow/outflow trends |
| **Stock Replenishment** | Reorder events, stock trends, products below reorder level, stockout alerts |
| **Sales Performance** | Revenue by region/channel, payment methods, quantity trends |
| **Order Fulfillment** | Order status, shipping performance, fulfillment rate, regional distribution |

---

## 📈 Key Insights

🔴 **Critical Issues:**
- **24.8% fulfillment rate** despite 100% shipment completion
- **73% revenue decline** from peak (GH₵49.2K → GH₵13.1K)
- **136 stockout events** despite adequate aggregate inventory

✅ **Strengths:**
- 100% shipment completion rate
- Balanced sales channels (51% online, 49% physical)
- Strong inventory management (only 1 product below reorder level)

---

## 🛠️ Tech Stack

- **Tool:** Microsoft Power BI Desktop
- **Data Model:** Star schema (4 fact tables, 5+ dimension tables)
- **DAX Measures:** 47+ custom calculations
- **Data Source:** Excel
